# 🚀 PathAI Pro - Advanced Learning Management System 

**PathAI Pro** is an intelligent, AI-driven learning platform designed to help students and professionals create customized learning paths. Instead of wandering through endless tutorials, users can generate a smart, step-by-step roadmap tailored to their specific goals. 

🔗 **[View Live Project Here](https://pathaiagent.ccbp.tech/)**

## ✨ Key Features

- **🗺️ Smart Roadmap Generation:** The AI performs deep research to generate a highly accurate and structured learning path based on the user's prompt.
- **💬 Live AI Mentor:** An integrated chatbot that acts as a personal mentor. Start your journey and ask any doubts in real-time.
- **📊 Progress Tracking:** Keep track of your learning journey with a dynamic status bar (0% to 100% completion).
- **📄 PDF Export:** Easily export your generated roadmap into a clean PDF format for offline access and printing.
- **⚡ Fast & Responsive UI:** A seamless user interface designed for an optimal learning experience.

## 🛠️ Tech Stack & Tools

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **AI Integration:** Generative AI APIs (LLM) for roadmap generation and live mentoring.
- **Backend/Automation:** *n8n*
- **Features:** Asynchronous API calls, DOM manipulation, PDF generation libraries.

## 🚀 How It Works

1. Enter your learning goal or the skill you want to master.
2. Click on **Generate Smart Roadmap**. The AI Agent will perform deep research.
3. Once generated, follow the step-by-step modules.
4. Interact with the **AI Mentor Live** if you get stuck or have doubts.
5. Click **PDF Export** to save your roadmap locally.
